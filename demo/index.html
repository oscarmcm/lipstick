Lipstick

Simple off-canvas navigation menu

v1.0.0


Why it's awesome?

Lipstick provides a minimal setup to show a side panel by sliding from the left or right of the page. Just it! Only 2kb gzipped!

Getting Started

There are some ways to get started. First, see all download options available below, then choose the most suitable option that you need.

Install with NPM

Run npm install lipstick, and link to ./node_modules/lipstick/dist/lipstick.min.js


Install with Bower

Run bower install lipstick, and link to ./bower_components/lipstick/dist/lipstick.min.js


Download the source code

Download Lipstick


Now you should add the main file of the Lipstick footer of your project. Just that! :)

What's included

Once downloaded, extract the compressed folder to see the main file in the uncompressed and minified version.

					lipstick/

How to use

Once you have included the library you need to give some settings to Lipstick

					$("#menu").lipstick({
	trigger: $(".trigger-lipst"),
	position: 'right',
	push: true,
	overlay: true,
	menuColor: '#dc6552'
});
