Lipstick

A simple off-canvas navigation menu
Currently v1.0.0

Getting Started
Why it's awesome?

Lipstick provides a minimal setup to show a side panel by sliding from the left or right of the page. Just it! Only 2kb gzipped!

Do you want to star it, tweet it, or share it with anyone? So do it! This means a lot to me ♥

Getting Started

There are some ways to get started. First, see all download options available below, then choose the most suitable option for your need. Now you should add the main file of the Milligram and the CSS Reset in the header of your project. Just that!

Download Lipstick

Install with Bower

Lipstick is available to install using Bower.

$ bower install lipstick

Install with npm

Lipstick is also available to install using npm.

$ npm install lipstick

What's included

Once downloaded, extract the compressed folder to see the main file in the uncompressed and minified version.

lipstick/
├── demo/
│   └── index.html
├── dist/
│   ├── lipstick.js
│   └── lipstick.min.js
├── LICENSE
└── README.md

Usage

First, use any method mentioned above to download Lipstick. Then, just add these tags in the head. Lipstick is also available via CDN.

Hello World!!




$("#menu").lipstick({
	trigger: $(".trigger-lipst"),
	position: 'right',
	push: true,
	overlay: true,
	menuColor: '#dc6552'
});
